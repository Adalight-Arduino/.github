## Fixing Common Adalight Setup Problems

- LEDs not responding? Recheck the Adalight Arduino upload, serial port selection, data pin, ground connection, and Adalight LED strip direction.  
- Wrong colors showing? Adjust RGB order in the Adalight software or Adalight code, then retest with a simple red, green, and blue calibration pattern.  
- Flickering or unstable output? Use a stronger power supply, improve grounding, reduce brightness, and confirm that Adalight FastLED or Adalight NeoPixel settings match the hardware.  
- Delayed lighting effects? Lower capture resolution, close background apps, and tune Adalight Prismatik or Adalight Hyperion refresh settings.  
- Raspberry Pi issues? Verify permissions, USB access, and service configuration before restarting the Adalight Raspberry Pi workflow.
